# Standardized Report Generator

## Instructions
- Collect inputs: report_title, audience, required_sections, page_length.
- Follow brand rules: formal tone, AP style, inclusive language.
- Always include sections: Executive Summary, Findings, Recommendations, Appendix.
- Keep within page_length.

## Example
- Input: “Draft a 2-page Q3 marketing performance report for executives.”
- Output: A report with those sections, concise bullets, and clear recommendations.
